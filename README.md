## ✨ Overview

**The Archive** is an experimental web experience designed as a **digital memory vault**.
It focuses on atmosphere, minimalism, and emotional interaction rather than traditional content-heavy design.

This project explores how interfaces can feel like **spaces**, not just websites.

---

## 🧠 Concept

* A **digital archive of fragments**
* Minimal UI with intentional emptiness
* Experience-driven design over information
* Inspired by terminal systems and futuristic interfaces

---

## 🛠️ Tech Stack

* **Framework:** Next.js
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion
* **Deployment:** Vercel

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/the-archive.git
cd the-archive
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Visit 👉 http://localhost:3000

---

## 🌍 Deployment

This project is deployed using **Vercel**

```bash
git push origin main
```

Automatic deployment is triggered on every push.
